# Python Quiz Application 📝
A console-based Python Quiz Application to test programming knowledge.  
Users can answer multiple choice questions and get instant feedback.

## 🚀 Features
- Store questions and options using dictionaries
- Check user answers with conditional statements
- Score calculation after quiz completion
- Multiple questions support

## 🛠 Technologies Used
- Python
- Dictionaries
- Conditional statements

## ▶ How to Run
1. Install Python
2. Run the file:
   python python_quiz_application.py
